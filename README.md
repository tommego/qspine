# qspine
spine runtime support for qt qml with spine-cpp 3.x

thanks for [spine-qml](https://github.com/irukandji/spine-qml)

screenshot:
![1](screenshots/1.png)
![2](screenshots/2.png)

currently support:
 - real time timescale
 - real time fps
 - real time animation mixing
 - real time skins 
 - real time animation
 - skeleton scale(force to reload)
 - async loading
 - debug bones
 - debug slots
